 # Basic config to fast run php7 applications with Docker

 Create your project in root context (e.g. laravel new myApp)

Will be available with this setup:

- PHP 7 FPM
- Nginx web server
- Mysql database
- Redis for cache

 ## Run
 ```sh
$ docker-compose up -d
```

## Be happy
```
http://localhost:8080/
```
